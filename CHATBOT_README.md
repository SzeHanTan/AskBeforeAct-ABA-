# 🤖 AI Chatbot for Learn Section

## Quick Overview

An intelligent AI assistant powered by Google Gemini 2.5 Flash that helps users learn about scams, summarize news, and analyze suspicious situations.

---

## ⚡ Quick Start

### For Users
1. Open Learn section in app
2. Tap "🤖 Ask AI" button (bottom right)
3. Start chatting!

### For Developers
```bash
# Already integrated! Just run:
cd askbeforeact
flutter pub get
flutter run -d chrome
```

---

## 🎯 What It Does

### 1. 📰 Summarize News
**One-click AI summary of all scam news**
- Tap "AI Summary" button in Latest News tab
- Get comprehensive analysis in seconds
- Understand trends and patterns
- Identify most dangerous scams

### 2. 💬 Answer Questions
**Ask anything about scams and fraud**
- "How to identify phishing?"
- "What are romance scam signs?"
- "How to protect from payment fraud?"
- Get clear, actionable answers

### 3. 🔍 Analyze Scenarios
**Verify if situations are scams**
- Describe suspicious message/offer
- AI evaluates scam likelihood
- Get red flags and recommendations
- Know what to do next

### 4. 💡 Provide Tips
**Get protection advice**
- Ask for specific scam type tips
- Get prevention strategies
- Learn best practices
- Stay protected
